认识微信小程序的基本设置

文件类型

以上为微信小程序的结构目录

微信小程序的文件类型有: .js、wxml、wxss、json

结合web前端来看可以方便更好理解

  • wxml —> html
  • wxss —> css
  • js —> js （emmmmmm……😄）
  • json —> json（就是一个普通的json文件……）

配置文件

json文件为页面的配置文件，app.json为全局配置文件，当我们生成页面的时候也会生成json文件，这个生成的json文件是当前页面的配置文件，其中的配置优先级高于全局的app.json配置文件，有冲突的会覆盖app.json中的配置，当然，app.json的配置与页面的json文件也有所不同

页面的创建

  1. 直接在pages下创建对应的js json wxml wxss文件
  2. 在app.json中的pages下创建路径，然后点击编译自动创建
  3. 在pages上右键选择新建Page，输入名称后自动创建对应的js json wxml wxss文件
